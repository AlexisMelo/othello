Coup IA_MinMax ( Couleur *  plateau,
Joueur  joueurAMaximiser,
int  profondeur 
)

Permet d'obtenir un Coup par une IA.

Le Coup est calculé automatiquement, la profondeur permet de spécifier combien de Coup à l'avance l'IA peut calculer son coup, sachant que plus ce nombre est élevé plus l'IA sera lente à répondre.

Paramètres
plateauPlateau de jeu
joueurAMaximiserJoueur dont on souhaite obtenir le meilleur coup possible
profondeurProfondeur de recherche de l'IA
Renvoie
Meilleur Coup obtenu par l'IA

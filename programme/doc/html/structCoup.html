Description détaillée

Le type Coup permet de jouer une couleur à une position.

Le Coup est composé d'une position et d'une couleur. Lorsque ce Coup est considéré comme joué, alors la couleur qu'il contient est insérée à la position donnée sur le plateau de jeu.
